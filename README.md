# Titanic Survival Pipeline

An end-to-end machine learning pipeline for predicting passenger survival on the Titanic dataset using Scikit-learn. This project emphasizes building reusable and maintainable ML pipelines by integrating data preprocessing, feature engineering, model training, evaluation, and prediction into a single workflow.

## Project Overview

The main goal of this project is to demonstrate how Scikit-learn Pipelines can streamline machine learning workflows by ensuring consistent transformations during both training and inference.

## Features

- Data preprocessing using Scikit-learn Pipelines
- Handling missing values
- Feature encoding and transformation
- Model training and comparison
- Hyperparameter tuning
- Reusable prediction pipeline
- Clean and maintainable workflow
